Hulucastable
This is a bookmarklet that allows you to open hulu links in a manner that makes it easy to cast hulu programs via chrome cast, rather than jump through a bunch of hoops.
Just drag the link above to your bookmarks bar, and click it on a hulu page. Any link thumbnail that has a red text background will now open it's video in the same window, using the full screen so you can use the chrome cast extension.
