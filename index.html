Bookmarklet

Hulucastable
This is a bookmarklet that allows you to open hulu links in a manner that makes it easy to cast hulu programs via chrome cast, rather than jump through a bunch of hoops.
Just drag the link above to your bookmarks bar, and click it on a hulu page. Any link thumbnail that has a green text background will now open it's video in the same window, using the full screen so you can use the chrome cast extension.

Chrome Extension

Hulucastable Extension
This is an extension that makes it so that every time you open hulu, it will automatically make hulu videos open in a manner appropriate for chromecasting.
Just click the link above to go install the extension to chrome, then open hulu and go to town streaming video from hulu to your chromecast.
